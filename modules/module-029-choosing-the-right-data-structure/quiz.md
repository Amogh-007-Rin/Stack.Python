# Quiz: Choosing the Right Data Structure

1. Which data structure is best for storing unique elements?
   - a) List
   - b) Tuple
   - c) Set
   - d) Dictionary

2. Which data structure provides O(1) average lookup by key?
   - a) List
   - b) Dictionary
   - c) Tuple
   - d) Both b and d

3. What is the time complexity of `in` for a list?
   - a) O(1)
   - b) O(log n)
   - c) O(n)
   - d) O(n²)

4. What is the time complexity of `in` for a set?
   - a) O(1)
   - b) O(log n)
   - c) O(n)
   - d) O(n²)

5. Which structure is best for an ordered collection that changes frequently?
   - a) Tuple
   - b) Set
   - c) List
   - d) Frozenset

6. Which structure is best for returning multiple values from a function?
   - a) List
   - b) Set
   - c) Tuple
   - d) Dictionary

7. If you need to look up a value by a unique identifier, which do you use?
   - a) List of tuples
   - b) Set
   - c) Dictionary
   - d) Tuple

8. Which data structure is NOT suitable for storing mutable elements?
   - a) List
   - b) Dictionary (as values)
   - c) Set
   - d) All can store mutable elements

9. For a fixed configuration that should never change after creation, which is best?
   - a) List
   - b) Dictionary
   - c) Tuple or namedtuple
   - d) Set

10. Which data structure would you use for a deck of playing cards where order matters and cards can repeat?
    - a) Set
    - b) List
    - c) Dict
    - d) Frozenset

**Answers:** 1-c, 2-b, 3-c, 4-a, 5-c, 6-c, 7-c, 8-c, 9-c, 10-b
