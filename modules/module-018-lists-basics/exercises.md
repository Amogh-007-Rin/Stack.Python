# Module 018: Lists: Basics — Exercises

## Exercise 1: List Create and Index
Create a list of 5 favorite foods. Print:
- The first item
- The last item (using negative indexing)
- The middle item

## Exercise 2: Slice Practice
Given `nums = [10, 20, 30, 40, 50, 60, 70, 80, 90]`, use slicing to get:
- First 3 elements
- Last 3 elements
- Every other element
- The list reversed

## Exercise 3: Sum and Average
Ask the user for 5 numbers, store them in a list, then calculate and print the sum and average.

## Exercise 4: Nested List Tic-Tac-Toe
Create a 3×3 tic-tac-toe board (nested lists) initialized with spaces. Place an "X" at (0,0), "O" at (1,1), and "X" at (2,2). Print the board.

## Exercise 5: Palindrome Checker
Ask the user for a word. Convert it to a list of characters. Check if the list equals its reverse (use slicing). Print "Palindrome" or "Not a palindrome".

## Exercise 6: List from String
Ask the user for a sentence. Use `list()` or a loop to create a list of characters. Count and print how many spaces are in the sentence.

## Exercise 7: Max and Min (No Built-in)
Given `values = [45, 12, 89, 3, 67, 24, 55]`, find and print the maximum and minimum values without using `max()` or `min()`. Use a loop.

## Exercise 8: List Shifting
Given `items = [1, 2, 3, 4, 5]`, "shift" it left by one position: `[2, 3, 4, 5, 1]`. Then "shift" it right by one: `[5, 1, 2, 3, 4]`. Do this with slicing.
