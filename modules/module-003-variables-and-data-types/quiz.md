## Answers

1. **C** — `answer = 42`
2. **False** — Hyphens are not allowed; use underscores
3. **B** — `<class 'float'>`
4. **`ten`** — The variable was reassigned to a string

6. **C** — bool
7. Dynamic typing means a variable can hold any type of value, and its type can change during program execution.
8. **C** — `class` is a reserved keyword
