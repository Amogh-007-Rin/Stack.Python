# Module 063: Modules and Imports

- **Phase:** 7. Modules, Stdlib & Testing
- **Duration:** 1.5 hours

## Learning Objectives

- Use the `import` statement, `from...import`, and `import...as`
- Understand the `__name__ == "__main__"` idiom
- Explore the module search path (`sys.path`)
- Create and import your own `.py` module files
- Handle circular imports and module reloading

## Topics Covered

- `import module` vs `from module import name`
- Aliasing with `import...as`
- `__name__` variable and guarding execution
- `sys.path` and how Python finds modules
- Writing and importing custom `.py` files
- `importlib.reload()` for live reloading
