# Module 063: Modules and Imports — Quiz

1. **What does `if __name__ == "__main__"` check for?**
   - a) Whether the module is imported
   - b) Whether the module is run directly
   - c) Whether `__name__` is a string
   - d) Whether the module is a package

2. **Which list does Python consult to find modules?**
   - a) `os.path`
   - b) `sys.modules`
   - c) `sys.path`
   - d) `os.modules`

3. **How do you import only the `sqrt` function from the `math` module?**
   - a) `import sqrt from math`
   - b) `from math import sqrt`
   - c) `import math.sqrt`
   - d) `use math.sqrt`

4. **What is the effect of `import mymodule as mm`?**
   - a) Renames the file to `mm.py`
   - b) Creates an alias `mm` for the module
   - c) Imports only the `mm` attribute
   - d) The import fails

5. **Which function reloads an already-imported module?**
   - a) `importlib.reload()`
   - b) `importlib.import()`
   - c) `reload()`
   - d) `module.refresh()`

<details>
<summary>Answers</summary>
1-b, 2-c, 3-b, 4-b, 5-a
</details>
