# Module 009: Exercises

## Exercise 1: Large Integers
Compute `2 ** 200` and print how many digits it has (hint: convert to string and use `len()`).

## Exercise 2: Float Pitfall
Predict the output:
```python
print(0.3 == 0.1 + 0.2)
```
Then explain why.

## Exercise 3: Safe Comparison
Write a function-like code snippet that checks if `0.1 + 0.2` equals `0.3` using a tolerance of `1e-10`.

## Exercise 4: math Module
Using the `math` module, compute and print:
- The square root of 144
- 3.14159 ceiling
- 3.14159 floor
- The value of π times e

## Exercise 5: Rounding
What does `round(4.5)` return? What about `round(5.5)`? Test and explain.

## Exercise 6: Geometry
Write code that computes the volume of a cylinder with radius 3 and height 7 using `math.pi`.

## Exercise 7: Complex Numbers
Create a complex number `a = 2 + 3j` and `b = 1 - 1j`. Print `a + b`, `a * b`, and `abs(a)`.

## Exercise 8: Euclidean Distance
Given two points `(x1, y1) = (1, 2)` and `(x2, y2) = (4, 6)`, compute the Euclidean distance using `math.sqrt()`. Distance formula: `sqrt((x2-x1)^2 + (y2-y1)^2)`.
