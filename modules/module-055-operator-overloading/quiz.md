# Module 055: Operator Overloading — Quiz

1. **Which method overloads the `+` operator?**
   - a) `__plus__`
   - b) `__add__`
   - c) `__sum__`
   - d) `__inc__`

2. **What should you return from `__eq__` when comparing with an unsupported type?**
   - a) `False`
   - b) `None`
   - c) `NotImplemented`
   - d) Raise `TypeError`

3. **Which decorator helps implement comparison operators with minimal code?**
   - a) `@dataclass`
   - b) `@total_ordering`
   - c) `@functools.cmp`
   - d) `@compare`

4. **What does `__abs__` overload?**
   - a) `abs()` function
   - b) `+` unary operator
   - c) `-` unary operator
   - d) `~` operator

5. **What is the return type convention for `__add__`?**
   - a) Always `None`
   - b) A new instance of the same class
   - c) A tuple
   - d) A string representation

<details>
<summary>Answers</summary>
1-b, 2-c, 3-b, 4-a, 5-b
</details>
