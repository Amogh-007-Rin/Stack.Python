# Module 067: Regular Expressions — Quiz

1. **Which function finds ALL non-overlapping matches in a string?**
   - a) `re.search()`
   - b) `re.match()`
   - c) `re.findall()`
   - d) `re.find()`

2. **What does `r"\d+"` match?**
   - a) One digit
   - b) One or more digits
   - c) A word character
   - d) A whitespace character

3. **Which metacharacter anchors a pattern to the start of a string?**
   - a) `$`
   - b) `^`
   - c) `\A`
   - d) `\b`

4. **What is the purpose of a raw string `r"..."` in regex?**
   - a) To make the regex case-insensitive
   - b) To avoid interpreting backslashes as escape sequences
   - c) To match raw text literally
   - d) To speed up matching

5. **How do you create a named capturing group?**
   - a) `(name:...)`
   - b) `(?P<name>...)`
   - c) `(?<name>...)`
   - d) `(:name ...)`

<details>
<summary>Answers</summary>
1-c, 2-b, 3-b, 4-b, 5-b
</details>
