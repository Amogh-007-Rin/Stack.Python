# Module 053: Dataclasses

- **Phase:** 6. Advanced OOP & Errors
- **Duration:** 1.5 hours

## Learning Objectives

- Define classes with the `@dataclass` decorator
- Automate `__init__`, `__repr__`, `__eq__`, `__hash__` generation
- Use `field()` for default values and factory functions
- Create immutable data classes with `frozen=True`
- Combine dataclasses with inheritance

## Topics Covered

- `@dataclass` decorator
- Auto-generated special methods
- `field()`: `default`, `default_factory`, `init`, `repr`
- Frozen (immutable) dataclasses
- Comparison with manually written classes
- Dataclass inheritance
