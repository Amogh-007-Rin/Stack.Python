# Module 036: Exercises

## Exercise 1: map with Function
Use `map()` with a named function (not lambda) to convert `["1", "2", "3"]` to integers.

## Exercise 2: filter for Negatives
Use `filter()` to keep only negative numbers from `[-3, -1, 0, 2, 5, -4]`.

## Exercise 3: reduce for Product
Use `reduce()` to calculate the product of all numbers in `[2, 3, 5, 7]`.

## Exercise 4: sorted by Custom Key
Sort `["banana", "apple", "cherry", "date"]` by the number of 'a's in each word.

## Exercise 5: Function Returning Function
Write a function `make_power(exp)` that returns a function raising its argument to `exp`.

## Exercise 6: map with Multiple Iterables
Use `map()` with two lists to add elements pairwise: `[1, 2, 3]` and `[10, 20, 30]`.

## Exercise 7: filter and map Chain
Given `["apple", "banana", "cherry"]`, first filter words with more than 5 letters, then map to uppercase.

## Exercise 8: Custom reduce
Write your own version of `reduce` called `my_reduce(func, iterable, initial=None)`.

## Exercise 9: sorted with Attribute
Given `people = [{"name": "Alice", "age": 30}, {"name": "Bob", "age": 25}]`, sort by age descending.

## Exercise 10: Compose Functions
Write a function `compose(f, g)` that returns a new function `h(x) = f(g(x))`.
