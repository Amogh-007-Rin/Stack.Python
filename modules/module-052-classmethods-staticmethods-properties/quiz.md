# Module 052: Class Methods, Static Methods, Properties — Quiz

1. **What is the first parameter of a `@classmethod`?**
   - a) `self`
   - b) `cls`
   - c) No parameters
   - d) `class`

2. **What is the first parameter of a `@staticmethod`?**
   - a) `self`
   - b) `cls`
   - c) No implicit parameters
   - d) `static`

3. **Which decorator is used for computed attributes?**
   - a) `@classmethod`
   - b) `@staticmethod`
   - c) `@attribute`
   - d) `@property`

4. **What does a property getter return?**
   - a) A class method
   - b) The computed attribute value
   - c) A static method
   - d) The class itself

5. **Why is `@property` better than manual getters/setters?**
   - a) It's faster
   - b) It allows attribute-style access without breaking encapsulation
   - c) It uses less memory
   - d) It automatically validates all data

<details>
<summary>Answers</summary>
1-b, 2-c, 3-d, 4-b, 5-b
</details>
