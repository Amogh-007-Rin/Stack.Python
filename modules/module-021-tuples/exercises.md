# Exercises: Tuples

## Exercise 1: Tuple Basics
Create a tuple `colors` containing `"red"`, `"green"`, `"blue"`. Print the first and last elements using indexing.

## Exercise 2: Immutability
Try to change the second element of a tuple `(10, 20, 30)` to `25`. Catch the resulting `TypeError` and print a message explaining why it failed.

## Exercise 3: Tuple Unpacking
Given `point = (4, 7)`, unpack it into variables `x` and `y` and print their sum.

## Exercise 4: Returning Multiple Values
Write a function `min_max(nums)` that returns the minimum and maximum of a list as a tuple. Test it on `[3, 1, 7, 2, 9]`.

## Exercise 5: Tuple as Dictionary Key
Create a dictionary mapping `(x, y)` coordinates to city names. Add at least three entries and look one up.

## Exercise 6: Namedtuple
Define a `namedtuple` called `Student` with fields `name`, `age`, `grade`. Create two students and print their names.

## Exercise 7: Tuple Methods
Given `t = (1, 2, 2, 3, 2, 4)`, use `.count()` and `.index()` to find how many times `2` appears and the index of the first `3`.

## Exercise 8: Slicing Tuples
Create a tuple of the numbers 1 through 10. Slice it to get:
- The first 5 elements
- Elements from index 3 to 7
- Every other element

## Exercise 9: Swapping with Tuples
Given `a = 5` and `b = 10`, swap their values using tuple unpacking (one line).

## Exercise 10: Nested Tuples
Create a nested tuple representing a 2x3 matrix `((1,2,3), (4,5,6))`. Access the element `5` and print it.
