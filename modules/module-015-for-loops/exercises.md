# Module 015: Loops: `for` — Exercises

## Exercise 1: Sum of Even Numbers
Use a `for` loop with `range()` to calculate the sum of all even numbers from 2 to 100 (inclusive).

## Exercise 2: Character Counter
Ask the user for a string and a character. Use a `for` loop to count how many times that character appears (case-sensitive).

## Exercise 3: Multiplication Table
Ask the user for a number. Use a `for` loop to print its multiplication table from 1 to 12.

## Exercise 4: Factorial with `for`
Calculate the factorial of a number using a `for` loop instead of `while`.

## Exercise 5: Reverse a String
Ask the user for a string. Use a `for` loop with `range()` to print it in reverse, one character per line.

## Exercise 6: FizzBuzz
Print numbers 1 to 50, but:
- For multiples of 3, print "Fizz" instead
- For multiples of 5, print "Buzz" instead
- For multiples of both, print "FizzBuzz"

## Exercise 7: `for-else` Prime Finder
Ask the user for a number. Use a `for-else` loop to determine if it's prime. Print "{n} is prime" or "{n} is not prime".

## Exercise 8: Sum of Digits
Ask the user for a positive integer. Use a `for` loop to sum its digits (hint: convert to string first).
