# Module 068: Working with Dates and Times — Quiz

1. **Which class represents a duration or difference between two datetimes?**
   - a) `datetime.duration`
   - b) `datetime.timedelta`
   - c) `datetime.diff`
   - d) `datetime.interval`

2. **What does `datetime.datetime.strptime("2025-01-01", "%Y-%m-%d")` return?**
   - a) A string
   - b) A `datetime` object
   - c) A `date` object
   - d) A `time` object

3. **Which module provides timezone support in Python 3.9+?**
   - a) `pytz`
   - b) `zoneinfo`
   - c) `timezone`
   - d) `tzlocal`

4. **What format code represents a full month name (e.g., "January")?**
   - a) `%m`
   - b) `%M`
   - c) `%b`
   - d) `%B`

5. **How do you add 7 days to a `datetime` object `dt`?**
   - a) `dt + 7`
   - b) `dt + timedelta(days=7)`
   - c) `dt.add(days=7)`
   - d) `dt.shift(days=7)`

<details>
<summary>Answers</summary>
1-b, 2-b, 3-b, 4-d, 5-b
</details>
