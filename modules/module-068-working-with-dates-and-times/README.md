# Module 068: Working with Dates and Times

- **Phase:** 7. Modules, Stdlib & Testing
- **Duration:** 1.5 hours

## Learning Objectives

- Use `date`, `time`, `datetime`, and `timedelta` classes deeply
- Handle time zones with `zoneinfo` (Python 3.9+)
- Perform date arithmetic and formatting
- Parse dates from strings using `strptime`

## Topics Covered

- `date`, `time`, `datetime`, `timedelta` classes
- Timezone handling with `zoneinfo.ZoneInfo`
- Date arithmetic with `timedelta`
- `strftime` / `strptime` format codes
- Parsing dates from arbitrary string formats
- Working with time deltas and durations
