# Exercises: Working with Collections

## Exercise 1: Basic zip()
Given `names = ["Alice", "Bob", "Charlie"]` and `scores = [85, 92, 78]`, use `zip()` to print `"name: score"` for each pair.

## Exercise 2: Unzipping
Given `pairs = [("x", 1), ("y", 2), ("z", 3)]`, use `zip(*pairs)` to separate into two tuples and print them.

## Exercise 3: enumerate() Basics
Given `fruits = ["apple", "banana", "cherry"]`, use `enumerate()` to print each fruit with its index (starting at 1).

## Exercise 4: enumerate() with Start
Repeat exercise 3 but with the index starting at 10.

## Exercise 5: itertools.count()
Use `itertools.count()` to generate the first 10 even numbers starting from 0.

## Exercise 6: itertools.cycle()
Use `itertools.cycle()` to repeat the sequence `["rock", "paper", "scissors"]` 5 times.

## Exercise 7: itertools.chain()
Use `itertools.chain()` to combine three lists `[1, 2]`, `[3, 4]`, `[5, 6]` into a single iterator and print all elements.

## Exercise 8: itertools.product()
Use `itertools.product()` to generate all combinations of `["A", "B"]` and `[1, 2, 3]`.

## Exercise 9: itertools.permutations()
Generate all 2-element permutations of `["a", "b", "c"]`.

## Exercise 10: itertools.combinations()
Generate all 2-element combinations of `["a", "b", "c"]`. Note how this differs from permutations.
