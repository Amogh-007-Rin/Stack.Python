# Module 058: Working with Files — Quiz

1. **What does `open("file.txt", "w")` do if the file already exists?**
   - a) Appends to it
   - b) Overwrites it
   - c) Raises FileExistsError
   - d) Opens in read-only mode

2. **Which mode prevents accidentally overwriting an existing file?**
   - a) `"w"`
   - b) `"a"`
   - c) `"x"`
   - d) `"r+"`

3. **What does the `with` statement guarantee?**
   - a) The file is compressed
   - b) The file is automatically closed
   - c) The file is encrypted
   - d) The file is backed up

4. **Which method reads all lines into a list?**
   - a) `.read()`
   - b) `.readline()`
   - c) `.readlines()`
   - d) `.readall()`

5. **What is the default encoding for `open()` in Python 3?**
   - a) ASCII
   - b) UTF-8
   - c) Latin-1
   - d) Platform-dependent (locale.getencoding())

<details>
<summary>Answers</summary>
1-b, 2-c, 3-b, 4-c, 5-d
</details>
