# Module 058: Working with Files: Reading & Writing Text Files

- **Phase:** 6. Advanced OOP & Errors
- **Duration:** 2 hours

## Learning Objectives

- Open files with `open()` and understand file modes
- Use the `with` statement as a context manager
- Read file contents with `.read()`, `.readline()`, `.readlines()`
- Write to files with `.write()` and `.writelines()`
- Iterate over a file line by line
- Specify encoding (especially UTF-8)
- Understand absolute vs relative file paths

## Topics Covered

- `open()`: `'r'`, `'w'`, `'a'`, `'x'`, `'r+'` modes
- Context manager: `with open(...) as f:`
- Reading methods: `.read()`, `.readline()`, `.readlines()`
- Writing methods: `.write()`, `.writelines()`
- File iteration (`for line in f`)
- `encoding='utf-8'`
- `os.path` and `pathlib.Path` basics
