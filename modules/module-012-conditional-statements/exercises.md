# Module 012: Conditional Statements (if / elif / else) — Exercises

## Exercise 1: Grade Classifier
Write a program that asks the user for a numeric grade (0–100) and prints the letter grade:
- 90–100: A
- 80–89: B
- 70–79: C
- 60–69: D
- Below 60: F
- Handle invalid input (outside 0–100)

## Exercise 2: Leap Year Checker
Write a program that asks for a year and prints whether it's a leap year.
A year is a leap year if it's divisible by 4, unless it's divisible by 100, unless it's divisible by 400.

## Exercise 3: Login Simulator
Create a simple login check. Pre-set a username and password. Ask the user for both. Check:
- If username is empty → "Username required"
- If password is empty → "Password required"
- If both match → "Welcome, {username}!"
- If username matches but password doesn't → "Wrong password"
- If username doesn't match → "User not found"

## Exercise 4: Nested Conditions
Write a program that asks for a person's age and whether they have a driver's license (y/n). Determine:
- Under 16: "Too young to drive"
- 16 or older but no license: "Get a license first"
- 16 or older with license: "You can drive"

## Exercise 5: Number Analyzer
Ask the user for a number. Print:
- "Positive" if > 0, "Negative" if < 0, "Zero" if 0
- "Even" or "Odd" (if non-zero)
- Example: "Positive and Even"

## Exercise 6: Simple Calculator
Ask the user for two numbers and an operator (+, -, *, /). Perform the operation and print the result. Handle:
- Division by zero
- Invalid operator
