# Exercises: Unpacking and Multiple Assignment

## Exercise 1: Basic Tuple Unpacking
Given `point = (10, 20)`, unpack into `x` and `y` and print both.

## Exercise 2: List Unpacking
Unpack the list `["Alice", 30, "Engineer"]` into `name`, `age`, `job` and print them.

## Exercise 3: Swapping Variables
Given `a = 5` and `b = 10`, swap their values using a single line without a temporary variable.

## Exercise 4: Extended Unpacking
Given `nums = [1, 2, 3, 4, 5]`, unpack so that `first = 1`, `middle = [2, 3, 4]`, `last = 5`.

## Exercise 5: Star at Beginning
Given `nums = [1, 2, 3, 4, 5]`, unpack so that `first, *rest = nums`. Print `first` and `rest`.

## Exercise 6: Throwaway Variable
Given `data = ("Alice", 30, "NYC", "555-1234")`, unpack only the name and city (ignore age and phone).

## Exercise 7: Unpacking Return Values
Define a function `split_name(full_name)` that returns `(first, last)`. Call it and unpack the result.

## Exercise 8: Unpacking in For Loops
Given `students = [("Alice", 85), ("Bob", 92), ("Charlie", 78)]`, use a for loop with unpacking to print each name and score.

## Exercise 9: Nested Unpacking
Given `nested = (1, (2, 3), 4)`, unpack so that `a = 1`, `b = 2`, `c = 3`, `d = 4`.

## Exercise 10: Ignoring with Star
Given `items = ["a", "b", "c", "d", "e"]`, unpack `first`, `second`, and `rest` (eating the middle elements).
