# Module 014: Loops: `while` — Exercises

## Exercise 1: Countdown
Write a program that counts down from 10 to 1 using a `while` loop, then prints "Blast off!".

## Exercise 2: Sum Until Zero
Write a program that repeatedly asks the user for numbers and adds them to a total. Stop when the user enters 0. Print the total.

## Exercise 3: Input Validation
Ask the user to enter a number between 1 and 10. Keep asking until they enter a valid number. Use `isdigit()` to validate.

## Exercise 4: Factorial
Calculate the factorial of a number `n` (n!) using a `while` loop. Example: `5! = 5 × 4 × 3 × 2 × 1 = 120`.

## Exercise 5: Guess the Number (Hard Mode)
The computer picks a random number 1–100. The user guesses. After each guess, print "Too high" or "Too low" or "Correct!". Count the number of guesses. Hard mode: limit to 7 guesses.

## Exercise 6: Average Calculator
Write a program that keeps reading numbers until the user enters 'done'. Then print the average (sum / count). Handle the case where no numbers were entered.

## Exercise 7: Multiplication Table Printer
Ask the user for a number `n`. Use a `while` loop to print the multiplication table for `n` from 1 to 12.
