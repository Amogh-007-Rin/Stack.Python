# Module 054: Enums — Quiz

1. **Which module contains the `Enum` class?**
   - a) `enums`
   - b) `enum`
   - c) `types`
   - d) `constants`

2. **What does `auto()` assign to enum members?**
   - a) Random values
   - b) Sequential integers starting from 1
   - c) The member name as a string
   - d) `None`

3. **Which enum type allows direct integer comparison?**
   - a) `Enum`
   - b) `IntEnum`
   - c) `IntFlag`
   - d) Both b and c

4. **Can you iterate over an Enum class?**
   - a) No
   - b) Yes, with `for member in MyEnum:`
   - c) Only with `list(MyEnum)`
   - d) Only with `MyEnum.members()`

5. **Why use enums over module-level constants?**
   - a) Type safety and namespacing
   - b) They use less memory
   - c) They're faster
   - d) They can be serialized automatically

<details>
<summary>Answers</summary>
1-b, 2-b, 3-b, 4-b, 5-a
</details>
