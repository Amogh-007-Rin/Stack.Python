# Module 054: Enums

- **Phase:** 6. Advanced OOP & Errors
- **Duration:** 1 hour

## Learning Objectives

- Define enumerated types with the `Enum` class
- Access enum members and their values
- Use `auto()` for automatic value assignment
- Work with `IntEnum` and `StrEnum`
- Iterate over enum members
- Choose enums over loose constants

## Topics Covered

- `from enum import Enum`
- Enum member syntax and access
- `auto()` for sequential values
- `IntEnum` and `StrEnum`
- Enum iteration and membership tests
- Enums vs module-level constants
