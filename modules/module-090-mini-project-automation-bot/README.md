# Module 090: Milestone Project: Personal Automation Bot / Mini API

- **Phase:** 9. Databases & Web Apps
- **Duration:** 3 hours
- **Milestone:** Yes

## Learning Objectives

- Build a FastAPI mini-API combined with automation features
- Implement file management endpoints (list files, organize downloads)
- Run automated tasks on a schedule
- Create system info endpoints
- Log all actions to a file
- Apply type hints, error handling, and API patterns
