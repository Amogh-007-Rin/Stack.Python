# Module 051: Composition vs Inheritance — Quiz

1. **What relationship does inheritance model?**
   - a) has-a
   - b) is-a
   - c) uses-a
   - d) contains-a

2. **What relationship does composition model?**
   - a) is-a
   - b) kind-of
   - c) has-a
   - d) behaves-as

3. **Which of the following is TRUE about composition?**
   - a) A subclass inherits all parent methods
   - b) An object contains another object as an attribute
   - c) It requires the `class Child(Parent)` syntax
   - d) It always replaces inheritance

4. **When should you prefer composition over inheritance?**
   - a) When there is a clear "is-a" relationship
   - b) When you need to reuse many methods from a base class
   - c) When the relationship is more "has-a" than "is-a"
   - d) When you want to use polymorphism

5. **What does "favor composition over inheritance" mean?**
   - a) Never use inheritance
   - b) Use composition when the relationship isn't clearly "is-a"
   - c) Inheritance is always wrong
   - d) Composition requires less code

<details>
<summary>Answers</summary>
1-b, 2-c, 3-b, 4-c, 5-b
</details>
