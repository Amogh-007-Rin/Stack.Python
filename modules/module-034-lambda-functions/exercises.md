# Module 034: Exercises

## Exercise 1: Basic Lambda
Write a lambda that returns the cube of a number. Test it with 3.

## Exercise 2: Lambda with Two Args
Write a lambda that returns the first argument raised to the power of the second.

## Exercise 3: Sort by Last Letter
Sort `["apple", "banana", "cherry", "date"]` by the last character using a lambda.

## Exercise 4: Sort by Second Element
Given `data = [(1, 3), (2, 1), (3, 2)]`, sort by the second element using a lambda.

## Exercise 5: Map with Lambda
Use `map()` with a lambda to add 10 to each number in `[5, 10, 15, 20]`.

## Exercise 6: Filter with Lambda
Use `filter()` with a lambda to keep only strings longer than 4 characters from `["hi", "hello", "hey", "greetings"]`.

## Exercise 7: Conditional Lambda
Write a lambda that returns "even" if a number is even and "odd" otherwise. (Hint: use conditional expression.)

## Exercise 8: Sorting Dicts
Given `people = [{"name": "Alice", "age": 30}, {"name": "Bob", "age": 25}]`, sort by age using a lambda.

## Exercise 9: Lambda Without Assignment
Write a lambda and call it immediately without assigning it to a variable.

## Exercise 10: Convert to Lambda
Convert this function to a lambda and test it:
```python
def starts_with_a(s):
    return s.lower().startswith("a")
```
