# Module 030: Milestone Project — Contact Book Application

**Phase:** 3. Data Structures  
**Estimated Time:** 3 hours  
**Milestone:** Yes

## Overview
Build a CLI contact book application that applies everything you've learned about data structures (dicts, lists, user input, loops). This is a milestone project — complete it before moving on.

## Topics
- CLI contact book application
- Add contact (name, phone, email)
- View all contacts
- Search by name
- Delete contact
- Update contact
- Dict for in-memory storage (name → info dict)
- Menu-driven loop

## Learning Objectives
- Build a complete CLI application from scratch
- Use dicts for structured in-memory storage
- Implement CRUD (Create, Read, Update, Delete) operations
- Design a user-friendly menu system
- Handle edge cases (duplicate names, missing contacts)

- All modules 000–029
