# Module 016: Loop Control (break, continue, loop else) — Exercises

## Exercise 1: Find First Even
Given a list of numbers `[3, 7, 9, 12, 15, 18]`, use a `for` loop with `break` to find and print the first even number.

## Exercise 2: Skip Vowels
Ask the user for a word. Print all consonants (skip vowels a, e, i, o, u) using `continue`.

## Exercise 3: Prime Checker with `for-else`
Write a program that asks for a number and uses `for-else` to determine if it's prime. Print the result.

## Exercise 4: Menu with Break
Create a simple menu with options: 1. Print numbers 1-10, 2. Print "Hello" 5 times, 3. Quit. Use a `while True` loop with `break`.

## Exercise 5: Nested Loop — Find a Pair
Write a program that finds the first pair of numbers `(i, j)` where `i` and `j` are between 1 and 10 and `i * j == 24`. Use nested loops with a flag to break out of both.

## Exercise 6: Filter with Continue
Given the string `"a1b2c3d4e5"`, extract only the digits using a loop and `continue`. Print the result as a string.

## Exercise 7: Pass Placeholder
Create a function stub that does nothing yet (use `pass`), then call it. Then fill it in to print "Function works!".
