# Exercises: Dictionaries: Basics

## Exercise 1: Creating a Dictionary
Create a dictionary `person` with keys `"name"`, `"age"`, `"city"` and appropriate values for yourself.

## Exercise 2: Accessing Values
Using the `person` dictionary, print the value of `"name"` using `[]`. Then print the value of `"country"` using `.get()` with a default of `"Unknown"`.

## Exercise 3: Adding and Updating
Start with `inventory = {}`. Add `"apples": 5`, `"bananas": 3`. Then update apples to `10`. Print the dictionary.

## Exercise 4: Iterating Keys
Given `scores = {"Alice": 85, "Bob": 92, "Charlie": 78}`, iterate over the keys and print each student's name.

## Exercise 5: Iterating Values
Using the same `scores` dict, iterate over values and print each score.

## Exercise 6: Iterating Items
Using the same `scores` dict, iterate over items and print `"name: score"` for each student.

## Exercise 7: Membership Check
Check if `"Alice"` is in `scores`. Check if `"David"` is in `scores`. Print appropriate messages.

## Exercise 8: Dictionary Length
Print the number of entries in the `scores` dictionary.

## Exercise 9: Word Lengths
Write a function `word_lengths(words)` that takes a list of strings and returns a dict mapping each word to its length.

## Exercise 10: Character Frequency
Write a function `char_count(text)` that returns a dictionary counting how many times each character appears in `text`. Ignore spaces.
