# Module 059: Working with Files: CSV — Quiz

1. **Which CSV reader returns rows as dictionaries?**
   - a) `csv.reader`
   - b) `csv.DictReader`
   - c) `csv.MapReader`
   - d) `csv.ObjectReader`

2. **What does `newline=""` do when opening a CSV file for writing?**
   - a) Removes all newlines
   - b) Prevents double line-spacing on Windows
   - c) Adds extra newlines
   - d) Converts to Windows line endings

3. **Which `csv.writer` method writes a single row?**
   - a) `.write()`
   - b) `.writerow()`
   - c) `.writerows()`
   - d) `.append()`

4. **What does `csv.QUOTE_ALL` do?**
   - a) Quotes no fields
   - b) Quotes all fields
   - c) Quotes only non-numeric fields
   - d) Quotes only fields containing special characters

5. **How do you specify a tab delimiter?**
   - a) `delimiter=tab`
   - b) `delimiter="\t"`
   - c) `sep=tab`
   - d) `delim="\t"`

<details>
<summary>Answers</summary>
1-b, 2-b, 3-b, 4-b, 5-b
</details>
