# Module 059: Working with Files: CSV

- **Phase:** 6. Advanced OOP & Errors
- **Duration:** 1.5 hours

## Learning Objectives

- Use the `csv` module for reading and writing CSV files
- Read CSV data with `csv.reader` and `csv.DictReader`
- Write CSV data with `csv.writer` and `csv.DictWriter`
- Handle headers, custom delimiters, and quoting
- Convert between CSV rows, lists, and dicts

## Topics Covered

- `csv.reader`: reading rows as lists
- `csv.writer`: writing lists as rows
- `csv.DictReader`: reading rows as dicts
- `csv.DictWriter`: writing dicts as rows
- `fieldnames`, `delimiter`, `quotechar`
- Handling headers automatically
