# Module 061: Working with JSON — Quiz

1. **Which function serializes a Python object to a JSON string?**
   - a) `json.load()`
   - b) `json.dumps()`
   - c) `json.stringify()`
   - d) `json.serialize()`

2. **What is the JSON counterpart of a Python `list`?**
   - a) object
   - b) array
   - c) string
   - d) number

3. **Which parameter to `json.dumps` enables pretty-printing?**
   - a) `pretty=True`
   - b) `indent=2`
   - c) `beautify=True`
   - d) `format=True`

4. **What happens when you call `json.dumps` on a `datetime` object without a custom encoder?**
   - a) It returns the ISO string automatically
   - b) It raises a `TypeError`
   - c) It returns the timestamp
   - d) It raises a `ValueError`

5. **Which approach handles a non-serializable type in `json.dumps`?**
   - a) `strict=True`
   - b) `default=str`
   - c) `skip=True`
   - d) `ignore=True`

<details>
<summary>Answers</summary>
1-b, 2-b, 3-b, 4-b, 5-b
</details>
