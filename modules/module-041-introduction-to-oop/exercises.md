# Module 041: Exercises

## Exercise 1: Define a Class
Define an empty `class Car`. Create an instance and print its type.

## Exercise 2: Class with Method
Define a `class Dog` with a method `bark(self)` that prints "Woof!". Create an instance and call `bark()`.

## Exercise 3: Class with `__init__`
Define a `class Person` with `__init__(self, name, age)`. Create two instances with different data.

## Exercise 4: Multiple Methods
Define a `class Rectangle` with `__init__(self, width, height)` and methods `area()` and `perimeter()`.

## Exercise 5: Instance Counter
Create a `class Counter` that increments a counter each time a method `increment()` is called. Track the count per instance.

## Exercise 6: Class Blueprint Analogy
Write a short explanation (as a comment) comparing a class to a blueprint and an object to a house built from it.

## Exercise 7: Multiple Instance Practice
Create a `class Student` with `__init__(self, name, grade)`. Create a list of 3 students and loop through them printing their names.

## Exercise 8: Method with Return
Define a `class Circle` with `__init__(self, radius)` and a method `circumference()` that returns the circumference.

## Exercise 9: Default Attribute
Define a `class Book` with `__init__(self, title, author, pages=100)`. Create two books with different page counts.

## Exercise 10: OOP vs Procedural
Write a function `create_student(name, grade)` that returns a dict, and compare it to using a Student class. Which is clearer and why?
