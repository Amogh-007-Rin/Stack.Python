## Question 1 (Multiple Choice)
What command checks your Python version?
- A) `python info`
- B) `python --version`
- C) `python check`
- D) `version python`

- A) Print a word document
- B) Print working directory
- C) Power down
- D) Parse web data

## Question 3 (Multiple Choice)

- A) `list`
- B) `show`
- C) `ls`
- D) `dir`

## Question 4 (True/False)
On macOS and Linux, you should type `python` (not `python3`) to run Python 3.
- True
- False

## Question 5 (Multiple Choice)
What is the difference between the REPL and script mode?
- A) There is no difference
- B) REPL runs one line at a time; script mode runs a whole file
- C) REPL is faster than script mode
- D) Script mode cannot use `print()`

## Question 6 (Multiple Choice)
What VS Code extension do you need for Python support?
- A) Python by Microsoft
- B) PyCharm Connector
- C) Pythonista
- D) Jupyter Extension Only

## Question 7 (Short Answer)
You are in `/home/user` and want to go to `/home/user/projects`. What command do you type?

## Question 8 (Multiple Choice)
What should you do before running a .py file after editing it?
- A) Close VS Code
- B) Save the file
- C) Restart the computer
- D) Delete the old file

---

## Answers

1. **B** — `python --version` (or `python3 --version`)
2. **B** — Print Working Directory
3. **C** — `ls`
4. **False** — Use `python3` on macOS/Linux; `python` may point to Python 2
5. **B** — REPL runs one line at a time; script mode runs a whole file
6. **A** — Python by Microsoft
7. **`cd projects`**
8. **B** — Save the file
