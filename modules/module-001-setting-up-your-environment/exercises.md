# Module 001: Exercises

## Exercise 1: Version Check
Write a Python script that prints your Python version using `sys.version`.

## Exercise 2: Terminal Simulation
Use Python's `subprocess` module (or just write the commands as comments) to simulate running `pwd`, `ls`, and `cd ..` in order.

## Exercise 3: Print from a Script
Write a short Python script that prints three lines:
- Your name
- Your favorite color
- What you want to learn in Python

## Exercise 4: Run a .py File
Create a file called `about_me.py` containing at least two `print()` statements. Then write the terminal command you would use to run it.

## Exercise 5: REPL vs Script
Explain in your own words: what is the difference between running `python3` interactively and running `python3 myfile.py`?

## Exercise 6: Directory Navigation
You are in `/home/student/projects`. Write the terminal command to:
a) Go up one directory
b) List what is there
c) Go into a folder called `python_course`

## Exercise 7: VS Code
What extension do you need to install in VS Code to get syntax highlighting and IntelliSense for Python?
