# Module 031: Exercises

## Exercise 1: Simple Function
Define a function `hello()` that prints "Hello, Python!" and call it.

## Exercise 2: Return Value
Define a function `double(n)` that returns `n * 2`. Call it with 7 and print the result.

## Exercise 3: Two Parameters
Define a function `subtract(a, b)` that returns `a - b`. Call it with 10 and 3.

## Exercise 4: Docstring
Add a Google-style docstring to the `subtract` function from Exercise 3.

## Exercise 5: Local vs Global
Predict the output of this code, then test:
```python
x = 10
def change_x():
    x = 20
    print("Inside:", x)
change_x()
print("Outside:", x)
```

## Exercise 6: Function with Loop
Write a function `sum_to(n)` that returns the sum of numbers from 1 to n using a loop.

## Exercise 7: Convert Temperature
Write a function `celsius_to_fahrenheit(c)` that converts Celsius to Fahrenheit using `(c * 9/5) + 32`.

## Exercise 8: Multiple Returns
Write a function `min_max(numbers)` that returns both the minimum and maximum from a list.

## Exercise 9: String Repeater
Write a function `repeat(text, times)` that returns the text repeated `times` times with spaces.

## Exercise 10: Even or Odd
Write a function `is_even(n)` that returns True if n is even, False otherwise.
