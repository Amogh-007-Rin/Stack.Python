# Module 066: The Python Standard Library Tour (os, sys, datetime, random)

- **Phase:** 7. Modules, Stdlib & Testing
- **Duration:** 2 hours

## Learning Objectives

- Navigate the filesystem with `os` module
- Access command-line arguments and system info with `sys`
- Work with dates and times using `datetime`
- Generate random values and shuffle sequences with `random`

## Topics Covered

- `os.getcwd()`, `os.listdir()`, `os.path.join()`, `os.path.exists()`, `os.mkdir()`, `os.environ`
- `sys.argv`, `sys.exit()`, `sys.version`, `sys.path`
- `datetime.datetime.now()`, `datetime.timedelta`, `strftime` / `strptime`
- `random.random()`, `random.randint()`, `random.choice()`, `random.shuffle()`, `random.sample()`
