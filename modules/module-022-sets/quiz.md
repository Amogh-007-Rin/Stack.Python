# Quiz: Sets

1. Which of these creates a set?
   - a) `s = {1, 2, 3}`
   - b) `s = set([1, 2, 3])`
   - c) Both a and b
   - d) `s = [1, 2, 3]`

2. What does `s = {}` create?
   - a) An empty set
   - b) An empty dictionary
   - c) An empty list
   - d) A syntax error

3. Which method removes an element from a set without raising an error if the element doesn't exist?
   - a) `.remove()`
   - b) `.delete()`
   - c) `.discard()`
   - d) `.pop()`

4. What is the union of `{1, 2}` and `{2, 3}`?
   - a) `{1, 2, 3}`
   - b) `{2}`
   - c) `{1, 3}`
   - d) `{1, 2, 2, 3}`

5. What does the `&` operator do on sets?
   - a) Union
   - b) Intersection
   - c) Difference
   - d) Symmetric difference

6. Which of the following can be elements of a set?
   - a) List
   - b) Dictionary
   - c) Tuple
   - d) Set

7. What is the output? `{x**2 for x in range(5)}`
   - a) `[0, 1, 4, 9, 16]`
   - b) `(0, 1, 4, 9, 16)`
   - c) `{0, 1, 4, 9, 16}`
   - d) `{0: 0, 1: 1, 2: 4, 3: 9, 4: 16}`

8. How do you create an empty set?
   - a) `s = {}`
   - b) `s = set()`
   - c) `s = []`
   - d) `s = ()`

9. What is a frozenset?
   - a) A mutable set
   - b) An immutable set
   - c) A set that only holds frozen values
   - d) A set with a fixed size

10. What does `{1, 2, 3} - {2, 3, 4}` return?
    - a) `{1}`
    - b) `{4}`
    - c) `{1, 4}`
    - d) `{1, 2, 3, 4}`

**Answers:** 1-c, 2-b, 3-c, 4-a, 5-b, 6-c, 7-c, 8-b, 9-b, 10-a
