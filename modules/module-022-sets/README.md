# Module 022: Sets

**Phase:** 3. Data Structures  
**Estimated Time:** 1.5 hours  
**Milestone:** No

## Overview
Learn about sets, an unordered collection of unique elements. Master set operations for efficient membership testing and data deduplication.

## Topics
- Creating sets with `{}` and `set()`
- Set methods: `add`, `remove`, `discard`, `pop`, `clear`, `copy`
- Set operations: union (`|`), intersection (`&`), difference (`-`), symmetric difference (`^`)
- Set comprehensions
- `frozenset` (immutable set)
- Eliminating duplicates from sequences

## Learning Objectives
- Create and manipulate sets using braces and the `set()` constructor
- Use set methods to add, remove, and manage elements
- Perform mathematical set operations (union, intersection, difference, symmetric difference)
- Write set comprehensions
- Explain when to use `frozenset`
- Use sets to eliminate duplicate values
