# Module 085: Milestone Project: Personal Finance Database App

- **Phase:** 9. Databases & Web Apps
- **Duration:** 3 hours
- **Milestone:** Yes

## Learning Objectives

- Build a complete SQLite-backed CLI application using SQLAlchemy
- Implement CRUD operations for financial transactions
- Generate spending reports by category
- Create a monthly summary with aggregation
- Apply type hints, error handling, and ORM patterns
