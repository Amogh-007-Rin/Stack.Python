# Module 007: Exercises

## Exercise 1: Indexing Practice
Given `word = "Programming"`, write code to extract:
- The first character
- The last character
- The 6th character (index 5)
- The second-to-last character

## Exercise 2: Slicing
Given `text = "Hello, World!"`, use slicing to get:
- `"Hello"`
- `"World"`
- `"!dlroW ,olleH"` (reversed)
- `"Hlo ol!"` (every 2nd character)

## Exercise 3: Palindrome Check
Write code to check if the string `"radar"` is a palindrome (reads same forwards and backwards).

## Exercise 4: String Methods
Given `data = "  python;PROGRAMMING;language  "`, use string methods to:
1. Strip whitespace
2. Convert to lowercase
3. Split by `";"`
4. Print each part capitalized

## Exercise 5: Character Count
Write code that counts how many times the letter `"e"` appears in `"The quick brown fox jumps over the lazy dog"`.

## Exercise 6: Immutability
What does this code print? Why?
```python
text = "hello"
text.upper()
print(text)
```

## Exercise 7: Escape Sequences
Write a print statement that outputs:
```
She said: "Python's great!"
```
on one line.

## Exercise 8: URL Extractor
Given `url = "https://www.example.com/blog/post-title"`, use slicing to extract:
- The protocol (`"https"`)
- The domain (`"www.example.com"`)
- The path (`"/blog/post-title"`)
