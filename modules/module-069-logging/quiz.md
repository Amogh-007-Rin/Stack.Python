# Module 069: Logging — Quiz

1. **Which is the highest severity logging level?**
   - a) DEBUG
   - b) ERROR
   - c) CRITICAL
   - d) FATAL

2. **Which function configures logging with a single call?**
   - a) `logging.config()`
   - b) `logging.basicConfig()`
   - c) `logging.setup()`
   - d) `logging.init()`

3. **Which handler writes log messages to the console?**
   - a) `FileHandler`
   - b) `StreamHandler`
   - c) `ConsoleHandler`
   - d) `OutputHandler`

4. **What is the default logging level when no configuration is set?**
   - a) DEBUG
   - b) INFO
   - c) WARNING
   - d) ERROR

5. **Which is a good reason to use logging over print?**
   - a) Logging is faster
   - b) Logging supports levels, formatting, and output routing
   - c) print only works in debug mode
   - d) Logging cannot be disabled

<details>
<summary>Answers</summary>
1-c, 2-b, 3-b, 4-c, 5-b
</details>
