# Module 065: Virtual Environments and pip — Quiz

1. **Which command creates a virtual environment named `venv`?**
   - a) `python create venv`
   - b) `python -m venv venv`
   - c) `pip venv create`
   - d) `virtualenv init venv`

2. **On Linux, how do you activate a virtual environment in the `venv` directory?**
   - a) `source venv/bin/activate`
   - b) `venv\Scripts\activate`
   - c) `activate venv`
   - d) `source venv/Scripts/activate`

3. **Which command generates a `requirements.txt` from the current environment?**
   - a) `pip list > requirements.txt`
   - b) `pip freeze > requirements.txt`
   - c) `pip export > requirements.txt`
   - d) `pip save > requirements.txt`

4. **What does `pip install -r requirements.txt` do?**
   - a) Installs packages listed in the file
   - b) Removes packages listed in the file
   - c) Updates pip itself
   - d) Creates a new venv

5. **Which statement about venv is true?**
   - a) venv is a third-party tool
   - b) venv is built into Python 3
   - c) venv only works on Linux
   - d) venv installs packages globally

<details>
<summary>Answers</summary>
1-b, 2-a, 3-b, 4-a, 5-b
</details>
