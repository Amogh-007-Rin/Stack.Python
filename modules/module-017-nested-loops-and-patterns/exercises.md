# Module 017: Nested Loops and Pattern Printing — Exercises

## Exercise 1: Rectangle
Print a 4×6 rectangle of `#` characters using nested loops.

## Exercise 2: Right Triangle (Numbers)
Print a right triangle where each row shows the row number repeated:
```
1
22
333
4444
55555
```

## Exercise 3: Inverted Pyramid
Print an inverted pyramid of stars with height 5:
```
*********
 *******
  *****
   ***
    *
```

## Exercise 4: Multiplication Grid
Print a 10×10 multiplication table with right-aligned numbers (3 characters wide each).

## Exercise 5: Hollow Rectangle
Print a 4×8 hollow rectangle — only the border is `*`, the interior is spaces.

## Exercise 6: Number Diamond
Print a diamond of numbers with width 5:
```
    1
   121
  12321
 1234321
123454321
 1234321
  12321
   121
    1
```

## Exercise 7: Times Table Memorizer
Use nested loops to print the times table from 1 to 12 in a grid format. Each cell shows "i×j=result" but only if j <= i (lower triangular).
