## Exercise 5
1. The **parser** reads the source file and checks for syntax errors, producing an Abstract Syntax Tree (AST).
2. The **compiler** translates the AST into platform-independent **bytecode** (stored in `.pyc` files in `__pycache__/`).
3. The **CPython Virtual Machine** reads and executes the bytecode on the host CPU.

This is often summarized as: *source → bytecode → interpreter*.
