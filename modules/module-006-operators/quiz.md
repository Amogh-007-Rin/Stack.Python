## Question 7 (Short Answer)
Write an expression that is `True` if `age` is between 13 and 19 inclusive.

## Question 8 (Multiple Choice)
What does `-10 // 3` return?
- A) -3
- B) -3.333
- C) -4
- D) 3

---

## Answers

1. **B** — 3
2. **C** — `==`
3. **16** (5+3=8, 8*2=16)
4. **A** — True (`True and False` is `False`, `not False` is `True`)
5. **C** — `%`
6. **B** — 11 (multiplication before addition)
7. **`13 <= age <= 19`**
8. **C** — -4 (floor division rounds down)
